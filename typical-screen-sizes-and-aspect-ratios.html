Typical screen sizes and aspect ratios

Posted on Пн 20 Ноябрь 2017 in Design

Wide 16:9

  • 16:9 = 1.777
  • 3840 x 2160 = 1.777
  • 2560 x 1440 = 1.777
  • 1920 x 1080 = 1.777
  • 1600 x x900 = 1.777
  • 1366 x x768 = 1.778
  • 1280 x 720 = 1.777
  • 960 x 540 = 1.777
  • 854 x 480 = 1.779
  • 640 x 360 = 1.777
  • 1216 x 684 = 1.777 (android's navigation drawer)

Wide 16:10

  • 16:10 = 1.6
  • 2560 x 1600 = 1.6
  • 1920 x 1200 = 1.6
  • 1280 x 800 = 1.6
  • 800 x 480 = 1.666

Non-wide 4:3

  • 4:3 = 1.333
  • 2048 x 1536 = 1.333
  • 1600 x 1200 = 1333
  • 1024 x 768 = 1.333
  • 800 x 600 = 1.333

Square

  • 1:1 = 1
  • 720 x 720 = 1 (Blackberry Classic 3,5")
  • 1440 x 1440 = 1 (Blackberry Passport 4,5")
  • 1080 x 1080 = 1
