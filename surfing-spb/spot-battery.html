Спот Бухта Батарейная

Описание

Спокойный спот, преимущественно с небольшими, но ровными волнами.

Спот располагается в одноименной бухте, которая делится пополам старым полуразрушенным причалом.

Лучшие условия для катания, как правило, в левой части спота - слева от причала если смотреть на него с берега.

Характеристики

  • Тип: Beach break / Reef break / Point break
  • Направление волны: Левая / Правая
  • Дно: Песок
  • Сложность: Начинающий / Продвинутый / Эксперт
  • Уровень загруженности: Низкий / Средний / Высокий
  • Угрозы: нет
  • Качество волн: Низкое / Среднее / Высокое

Расположение

Координаты: (59.971340, 29.125711)

Дополнительно

Спот работает при юго-западном ветре
