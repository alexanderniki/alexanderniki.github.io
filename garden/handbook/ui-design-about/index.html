Проектирование (дизайн) пользовательского интерфейса

Проектирование пользовательского интрфейса (UI design) - это процесс создания интерфейса программы или устройства обычно с упором на визуальную составляющую (внешний вид, стиль). Изредка может использоваться термин "user interface engineering"

Дизайн пользовательского интерфейса - часть дизайна пользовательского опыта (UX design).

Дизайн можно рассматривать как:

  • процесс проектирования
  • результат проектирования - способ решения поставленной проблемы и артефакты его проектирования
